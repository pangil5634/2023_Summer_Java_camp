# E01 - 입력한 숫자 크기의 정사각형 출력

숫자를 하나 입력받은 후에 이 숫자만큼의 크기를 갖는 정사각형을 '#' 문자로 화면에 출력하라. 예를 들어 10을 입력하면 10개의 '#' 문자가 들어있는 라인 10개를 출력하는 것이다. 

변수는 다음과 같이 사용하라. 

int length; // 입력받은 정사각형 한 변의 길이 

int i, j; // 반복문 사용을 위한 변수
